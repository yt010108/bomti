# 제품 기획

## 제품 정의

Bomti는 AI 에이전트가 실제 업무를 얼마나 잘 수행하는지 평가하고, 사람이 선호한 좋은 결과물을 학습 데이터로 축적하는 플랫폼이다.

## 초기 포지션

공공기관·IT·보안 취업 준비 업무를 수행하는 한국어 에이전트 평가 플랫폼.

## 첫 사용자

- 공공기관 IT·보안 직무 취업 준비생
- 자기소개서와 프로젝트를 직무 경험으로 정리하려는 사람
- 면접 준비 자료를 자동화하고 싶은 사람
- 한국어 업무형 에이전트 평가 데이터를 만들고 싶은 개발자

## 첫 번째 기능 범위

| 구분 | 포함 |
| --- | --- |
| 입력 | 기관, 직무, 공고, 프로젝트 설명, 자기소개서 초안 |
| 생성 | 에이전트 답변 2개 이상 생성 |
| 평가 | 루브릭 기반 점수와 근거 생성 |
| 선호 | 사용자가 더 나은 답변 선택 |
| 저장 | task, answer, judge_result, preference 저장 |
| 내보내기 | JSONL export |

## 제외할 것

- 실제 채용 매칭
- 기업 계정 기능
- 자동 지원 기능
- 복잡한 RL 학습
- 대규모 크롤링
- 결제 기능

## 성공 기준

- 샘플 task 20개를 평가할 수 있다.
- 같은 task에 대해 2개 이상의 답변을 비교할 수 있다.
- judge 평가와 사람 선호를 함께 저장할 수 있다.
- 결과를 JSONL로 내보낼 수 있다.
- 데모에서 평가 → 선호 → 데이터셋화 흐름이 보인다.
