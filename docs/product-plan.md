# 제품 정의

Bomti는 취업 준비생이 자기소개서 문항과 답변, 목표 직무, 회사·공고 맥락을 제공하면 그 답변의 맥락 불일치·상투성·신뢰 저해 위험을 평가하는 독립 Judge 서비스다.

## 사용자와 가치

| 항목 | 결정 |
| --- | --- |
| 첫 사용자 | 한국어 자기소개서·면접을 준비하는 취업 준비생 |
| 공개 경험 | 하나의 맥락화된 답변을 평가하고 0–100 Bomti 지수와 문장 근거를 제공 |
| 비로그인 | 브라우저/IP 기준 하루 한 번, 저장하지 않는 제한된 DeepSeek 미리보기 |
| 인증 사용자 | Google 로그인 후 캠페인당 세 번의 Luna·Terra 상세 평가와 삭제 가능한 이력 |
| 내부 보정 | 익명화된 별도 복사본을 이용한 페어 비교·오프라인 CLI 내보내기 |

## 결과의 의미

점수가 높을수록 답변이 더 밤티다. 구간은 `밤티 거의 없음`, `살짝 밤티`, `꽤 밤티`, `밤티 그 자체`이며 합격·불합격을 뜻하지 않는다. 결과는 다섯 위험 차원, 문장 세그먼트 근거, 설명, 제한된 개선 예시로 구성하고 완성 답변을 생성하지 않는다.

## 포함과 제외

포함 범위는 동의된 단일 답변 평가, 가명처리, 인증 이력/삭제, 내부 익명 벤치마크, 비용·쿼터 제어다. 제외 범위는 전체 재작성, 공개 A/B, 관리자 대시보드, 결제, 자동 지원, 크롤링, 채용 매칭, RL 학습이다.

정확한 제품 요구사항과 증거 기준은 [requirements.md](requirements.md)를 정본으로 사용한다.
