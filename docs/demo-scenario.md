# 데모 시나리오

## 제목

맥락을 잃은 자기소개서 답변의 Bomti 지수 확인

## 입력

| 항목 | 예시 |
| --- | --- |
| 질문 | 지원 동기와 직무 역량을 설명해 주세요. |
| 답변 | 보안에 관심이 많고 항상 최선을 다하겠습니다. |
| 목표 직무 | 정보보호 진단·분석 |
| 회사/공고 맥락 | 공공기관 정보보호 직무의 취약점 점검과 보안 운영 |
| 경험 근거 | 선택 항목으로만 제공하며 직접 식별 정보는 제출하지 않는다. |

## 흐름

1. 사용자는 필수 입력과 각 동의 항목을 확인한다.
2. Bomti는 답변을 문장 세그먼트로 분리하고 가명처리한다.
3. 게스트는 제한된 미리보기, 인증 사용자는 교차검증 평가를 요청한다.
4. 결과는 높은 점수일수록 더 밤티인 지수, 다섯 위험 차원, 근거 문장, 개선 방향을 표시한다.
5. 인증 결과만 삭제 가능한 이력으로 저장되며, 익명 벤치마크는 별도 보수적 비식별 검사를 통과한 복사본만 사용한다.

이 시나리오는 공개 답변 비교나 완성 답변 생성을 포함하지 않는다.
